Inventory management system based on GPT-4o-mini API. 

To use:
1. install dependencies listed in requirements_v4.txt
2. activate virtual environment 'source venv/bin/activate' (may not be required if dependencies not installed in venv)
3. Provide an API key ‘export OPENAI_API_KEY=__pastekeyhere__’
4. Start the script ;python src/version7.py'

This repository includes the json/inventory_full.json but if you wanted to create your own new inventory management system
from a new inventory .xlsx, you must run scripts/build_json.py. This step is not necessary for this repo.
